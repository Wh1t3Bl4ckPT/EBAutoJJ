Antes de tudo abre o ficheiro cmd para instalar os pacotes necessários para o script.
Depois ao instalar vai ser perguntado qual treino quer para Praça ou Graduados depois só escolher e vai começar.
Depois de entrar no jogo é só apertar F5 e vai ser digitado os JJ / Polichinelos automaticamente.
Graduados - 100 JJ
Praças - 50 JJ
Para parar o script é só fechar o cmd.
